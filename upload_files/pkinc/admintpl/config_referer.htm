Referrer aufzeichnen
Wenn diese Option aktiviert ist, werden die Herkunftsseiten Ihrer Besucher aufgezeichnet. Speicherzeitraum
Referrer bleiben für die Anzahl der eingetragenen Tage gespeichert. Tragen Sie 0 ein, wird die automatische Löschung deaktiviert. Referrer filtern
URL oder Teile einer URL die durch die Aufzeichnung der Referrer nicht erfaßt werden sollen. Bereits erfasste Adressen, auf die diese Einstellungen zutreffen, werden bei der Anzeige ausgefiltert.
Mehrere Angaben möglich, je Zeile nur eine Angabe.
