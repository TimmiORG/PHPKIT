Benutzer- und Nicknamen
Legt fest welche Benutzer- und Nickname nicht gewählt werden können.
Hinweis zur Verwendung:
  • du sperrt alle Übereinstimmungen
  • für die Sperrung einzelner Namen verwenden Sie die Angabe: {du}
Je Zeile nur eine Angabe. E-Mail-Adressen
Tragen Sie hier die E-Mail-Adessen ein, die nicht verwendet werden dürfen.
Verwenden Sie * als Platzhalter um eine Domain zu bannen.
(Bsp: *@domain.de - sperrt alle domain.de E-Mail-Adressen)
Je Zeile nur eine Angabe. IP-Adressen sperren
Tragen Sie hier die IP-Adessen ein, die Sie sperren möchten. Verwenden Sie * als Platzhalter um IP-Bereiche zu sperren.
Je Zeile nur eine Angabe. Textzensur
Tragen Sie hier die Begriffe ein, die automatisch zensiert werden sollen.
Hinweis zur Verwendung:
  • und ersetzt alle Übereinstimmungen, z.B. Hund wird zu H***
  • für Einzelwörter verwenden Sie die Angabe: {und}
Je Zeile nur eine Angabe.
